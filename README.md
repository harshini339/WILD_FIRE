# Wildfire Detection Using Deep Learning

## Description

Wildfire Detection Using Deep Learning is an AI-based project developed to detect wildfire occurrences from images in real time. The system uses deep learning techniques to identify fire and smoke at an early stage, helping reduce response time and minimize damage to forests, wildlife, and nearby communities.

The model is trained using image datasets and applies computer vision techniques for accurate detection. By automating the monitoring process, the project provides a faster and more reliable alternative to manual surveillance.

## Features

- Real-time wildfire detection from images
- Deep learning-based fire and smoke classification
- Image preprocessing for improved accuracy
- Early detection to support quick emergency response
- User-friendly and scalable solution

## Technologies Used

- Python
- TensorFlow
- OpenCV
- CNN (Convolutional Neural Networks)
- YOLOv11
- NumPy

## How It Works

1. The system takes an input image or video frame.
2. Image preprocessing is performed to enhance image quality.
3. The trained deep learning model analyzes the input.
4. The model detects the presence of wildfire or smoke.
5. The detected results are displayed, enabling timely action.

## Conclusion

This project demonstrates the application of Deep Learning and Computer Vision in environmental monitoring. By enabling early wildfire detection, it can assist authorities in taking faster action, reducing the impact of forest fires and protecting natural resources.
